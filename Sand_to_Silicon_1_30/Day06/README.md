# Day 06 — Latches and Flip Flops
## Files
| File | Description |
|------|-------------|
| `sr_latch.v` | SR Latch using cross-coupled NOR gates |
| `d_latch.v` | D Latch — transparent latch |
| `d_flipflop.v` | D Flip Flop — edge triggered |
| `jk_flipflop.v` | JK Flip Flop — toggle state verified |
## Key Learning
- Latch is level-sensitive, Flip Flop is edge-triggered
- SR Latch has forbidden state — JK eliminates it
- D Flip Flop is fundamental building block of all registers
